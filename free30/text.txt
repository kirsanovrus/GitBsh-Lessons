Commands in the lesson: 

* pwd - узнать, где ты
* cd - смена папки 
* ls -la - список всех файлов
* touch lessontext - создать файл
* vim - открыть файл в строке
* git status - статус
* git add . 
  git add file - добавить файл 
* git commit -m 'text of message' - закоминить 
* git push - запушить изменения во внешний репозиторий 
Последние сообщения не пушаться. 
* git fetch - проверка изменений с внешних репозиториев 
* git pull
